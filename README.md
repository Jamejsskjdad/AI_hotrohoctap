✅ Yêu cầu môi trường
Node.js ≥ 18 và npm
Hệ điều hành bất kỳ (Linux/Mac/Windows)
(Tùy chọn) Git để clone repo
⬇️ Cài đặt nhanh
cd backend && npm install
cd ../frontend && npm install
▶️ Chạy dự án
Mở 2 terminal:
Terminal 1 – Backend
cd backend
node server.js
# Server sẽ lắng nghe ở PORT (mặc định 8787)
Terminal 2 – Frontend
cd frontend
npm run dev
# Vite sẽ chạy ở http://localhost:5173 (mặc định)
